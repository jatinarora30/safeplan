\doxysubsection{Detailed Description}
Weighted A$\ast$ planner for N-\/dimensional occupancy grids. 

Implements Weighted A$\ast$ on an N-\/D Num\+Py occupancy grid where value 0 denotes free space and value 1 denotes an obstacle. The planner accepts start and goal grid indices and returns a collision-\/free path if one exists. Weighted A$\ast$ scales the heuristic by a user-\/specified weight to favor greedy expansions and reduce runtime at the cost of potential suboptimality when the weight exceeds 1.

This implementation expands a full 3$^\wedge$N -\/ 1 neighborhood (axis-\/aligned and diagonal moves). Axis-\/aligned steps use unit cost; off-\/axis steps use the Euclidean length of the step vector. A priority queue orders expansions by g(n) + w·h(n), where g(n) is the accumulated path cost and h(n) is the chosen heuristic (Manhattan in index space).

Neighborhood generation is provided by adjacent\+Coordinates().

\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item Coordinates are handled in grid index space.
\item Step costs\+: axis steps cost 1.\+0; diagonal/off-\/axis steps use the Euclidean length of the index step.
\item The neighborhood includes all immediate offsets except the zero vector.
\item Heuristic used here is Manhattan distance; adjust if you change the move set.
\item With w $>$ 1, the solution is not guaranteed to be optimal.
