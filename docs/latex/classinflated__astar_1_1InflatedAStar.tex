\doxysubsection{Detailed Description}
A$\ast$ planner that runs on an EDT-\/inflated occupancy grid. 

\begin{DoxyParagraph}{Pipeline}
1) Compute EDT on (1 -\/ grid) so free cells carry positive clearance. 2) Inflate obstacles by thresholding EDT at {\ttfamily radius\+Inflate}. 3) Run A$\ast$ on the inflated grid with full 3$^\wedge$N -\/ 1 neighborhood. 4) Reconstruct the path if the goal is reached.
\end{DoxyParagraph}
This strategy is a simple way to enforce a clearance margin using standard A$\ast$.
