\doxysubsection{Detailed Description}
A$\ast$ with EDT-\/weighted safety costs (no hard inflation). 

\begin{DoxyParagraph}{Pipeline}
1) Compute EDT on (1 -\/ grid) so free cells carry clearance values. 2) Use full 3$^\wedge$N -\/ 1 neighborhood for expansion. 3) Edge cost = step\+\_\+cost + safety\+Factor / (epsilon + avg(EDT(u), EDT(v))). 4) Run A$\ast$ and reconstruct the path on success.
\end{DoxyParagraph}
This preserves reachability of narrow corridors while penalizing low-\/clearance motion, often yielding safer paths compared to pure geometric cost.
