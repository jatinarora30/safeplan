\doxysubsection{Detailed Description}
Dijkstra planner for N-\/dimensional occupancy grids. 

Implements Dijkstra’s shortest-\/path search on an N-\/D Num\+Py occupancy grid where value 0 denotes free space and value 1 denotes an obstacle. The planner accepts start and goal grid indices and returns a collision-\/free path with minimum accumulated cost if one exists.

This implementation expands a full 3$^\wedge$N -\/ 1 neighborhood (axis-\/aligned and diagonal moves). Axis-\/aligned steps use unit cost; diagonal or off-\/axis steps use the Euclidean length of the step vector. A priority queue orders expansions by the current best distance from the start. No heuristic is used.

Neighborhood generation is provided by adjacent\+Coordinates().
