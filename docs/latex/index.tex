 \hypertarget{index_autotoc_md1}{}\doxysection{Overview}\label{index_autotoc_md1}
{\bfseries{Safe\+Plan}} is a benchmarking suite for evaluating classical and modern path planning algorithms with a strong focus on {\bfseries{safety and optimality}} in robotics. ~\newline
 It compares traditional planners with safety-\/aware planners, enabling fair evaluation of how algorithms balance safety and efficiency. ~\newline


Safe\+Plan also includes the proposed {\bfseries{Opti\+Safe}} metric, designed to quantify the trade-\/off between path optimality and safety, providing a more realistic assessment of planner performance in safety-\/critical robotic applications. ~\newline
\hypertarget{index_autotoc_md2}{}\doxysection{Key Features}\label{index_autotoc_md2}

\begin{DoxyItemize}
\item {\bfseries{Batch experiments at scale}}
\begin{DoxyItemize}
\item Run thousands of randomized start‚Äìgoal trials across multiple maps automatically.
\item Supports repeatable benchmarking with structured JSON outputs.
\end{DoxyItemize}
\item {\bfseries{Reproducibility \& logging}}
\begin{DoxyItemize}
\item Standardized JSON logs for each run with environment, path, and evaluation info.
\item Aggregated statistics in CSV/\+JSON and optional visualization outputs.
\end{DoxyItemize}
\item {\bfseries{Visualization tools}}
\begin{DoxyItemize}
\item 2D plots of environments and paths.
\item 3D interactive visualization (via Plotly) for voxel maps and trajectories.
\end{DoxyItemize}
\item {\bfseries{Efficient \& stable execution}}
\begin{DoxyItemize}
\item Each run is sandboxed in subprocesses for memory isolation and reliability.
\item Lightweight and scalable to large benchmark runs.
\end{DoxyItemize}
\item {\bfseries{Extensible framework}}
\begin{DoxyItemize}
\item Add new planners, environments, or metrics with minimal boilerplate.
\item Designed for all kinds of classical planners, sampling based and geometry based.
\end{DoxyItemize}
\end{DoxyItemize}\hypertarget{index_autotoc_md3}{}\doxysection{Evaluation Metrics}\label{index_autotoc_md3}

\DoxyCodeLine{}

\DoxyCodeLine{sf = SafePlan("{}/home/run1.json"{})}
\DoxyCodeLine{sf.benchmark()}
\DoxyCodeLine{}
\DoxyCodeLine{\# 2) Compute aggregate statistics across all outputs}
\DoxyCodeLine{st = Stats("{}/home/run1.json"{})}
\DoxyCodeLine{st.compute()}
\DoxyCodeLine{}
\DoxyCodeLine{\# 3) Visualize one iteration (2D ‚Üí PNG, 3D ‚Üí Plotly HTML)}
\DoxyCodeLine{viz = Visualize()}
