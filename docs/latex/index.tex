 {\bfseries{Safe\+Plan}} is a benchmarking suite for evaluating classical and modern path planners in robotics with a strong emphasis on {\bfseries{safety, smoothness, and optimality}}. ~\newline
 Unlike existing frameworks (e.\+g., Path\+Bench), which primarily measure path cost and runtime, Safe\+Plan introduces {\bfseries{safety-\/critical and dynamics-\/aware metrics}} that better reflect the requirements of real robotic systems.

\item {\bfseries{Path Cost (m)\+:}} ~\newline
 Total geometric length of the path in meters, based on Euclidean distance between waypoints. Lower cost indicates a shorter and more efficient route. ~\newline

\item {\bfseries{Planning Time (ms)\+:}} ~\newline
 The time taken by the planner to generate a path, measured in milliseconds. Faster planners achieve lower values. ~\newline

 Percentage of planning attempts that successfully reach the goal. Higher success rate reflects reliability of the algorithm. ~\newline

\item {\bfseries{Distance to Goal (m, failures)\+:}} ~\newline
 Distance from the last explored path node to the goal when planning fails. Smaller values show the planner got closer to success. ~\newline

\item {\bfseries{Nodes in Path\+:}} ~\newline
 Number of waypoints in the final path. Fewer nodes suggest a smoother, simpler, and more direct path. ~\newline

\item {\bfseries{Peak Memory (KB)\+:}} ~\newline
 Maximum RAM used by the planner during execution. Lower values indicate more memory-\/efficient planning. ~\newline
