\doxysubsection{Detailed Description}
Abstract base/registry for evaluating planning algorithms. 

\begin{DoxyParagraph}{Responsibilities}

\begin{DoxyItemize}
\item Defines the {\ttfamily \mbox{\hyperlink{classbaseeval_1_1BaseEval_aca0c3fd526aa2e27ba129f3c452fcad8}{eval()}}} interface used by all evaulation metrices classes.
\item Establishes the common return contract {\ttfamily value} for that metrices.
\end{DoxyItemize}
\end{DoxyParagraph}
