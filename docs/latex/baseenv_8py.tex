\doxysubsection{Detailed Description}
Base class for environment generation used by planning algorithms. 

Provides a common interface for environment builders that produce N-\/dimensional occupancy grids and associated metadata. Subclasses implement getmap() to return a grid and start/goal pairs in a consistent format consumed by Safe\+Plan.
