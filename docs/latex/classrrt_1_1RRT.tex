\doxysubsection{Detailed Description}
\mbox{\hyperlink{classrrt_1_1RRT}{RRT}} planner operating on N-\/D occupancy grids. 

The planner maintains a tree of explored, collision-\/free grid indices. Each iteration\+: 1) Sample a random node with goal bias (see \mbox{\hyperlink{classrrt_1_1RRT_a4c1e29f53ad9fe872d734b151a9efe30}{get\+Random\+Node()}}). 2) Find the nearest existing node in the tree (see \mbox{\hyperlink{classrrt_1_1RRT_a4b127c6da77ab51c2aeef3dcda78bf24}{get\+Nearest\+Node()}}). 3) Steer from the nearest node toward the random sample by at most the step size. 4) If the edge is collision-\/free (see \mbox{\hyperlink{classrrt_1_1RRT_ac9212c57e97017d0fbcec815774c86ab}{is\+Edge\+Free()}}), add the new node. 5) If the new node is close enough to the goal and the final edge is free, declare success and reconstruct the path.
