\doxysubsection{Detailed Description}
Utilities to generate N-\/dimensional occupancy grids with polygonal obstacles and optional random start/goal pairs. 

Reads a scene configuration, rasterizes polygon obstacles into a discrete grid, and (optionally) samples random free start/goal pairs. Caches the generated grid to a JSON file for quick reloads.
