\doxysubsection{Detailed Description}
Utility to visualize 2D and 3D path planning results. 

This class loads a run configuration JSON file and corresponding algorithm iteration files to plot planned paths in either grid-\/based (2D) or voxel-\/based (3D) environments.

The plots display\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{-\/ Obstacles (black)}

\end{DoxyCode}

\begin{DoxyItemize}
\item Free space (white)
\item Algorithm-\/generated paths (colored lines)
\item Start and goal positions (highlighted markers) 
\end{DoxyItemize}
