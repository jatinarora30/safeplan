\doxysubsection{Detailed Description}
A$\ast$ planner for N-\/dimensional occupancy grids. 

Implements A$\ast$ (A-\/star) search over an N-\/D Num\+Py occupancy grid where 0 denotes free space and 1 denotes an obstacle. The planner takes a start and goal index (tuples of ints) and returns a collision-\/free path if one exists. The heuristic and neighborhood definition are provided by heuristics() and adjacent\+Coordinates() and must be chosen consistently.

\item {\ttfamily goal} \+: tuple\mbox{[}int, ...\mbox{]} — goal grid cell
\item {\ttfamily grid} \+: numpy.\+ndarray (N-\/D), values \{0=free, 1=obstacle\}
\end{DoxyItemize}

\item 8-\/neighbors / diagonal moves → Chebyshev (or Euclidean with proper step costs)
