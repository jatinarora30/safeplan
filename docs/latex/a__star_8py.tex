\doxysubsection{Detailed Description}
N-\/dimensional A$\ast$ (A-\/star) planner over Num\+Py occupancy grids. 

This module implements A$\ast$ search on an N-\/D occupancy grid where {\ttfamily 0} denotes free space and {\ttfamily 1} denotes an obstacle. The planner accepts start and goal grid indices (tuples of ints) and returns a collision-\/free path if one exists.


\begin{DoxyItemize}
\item Heuristic\+: By default, this implementation uses the {\bfseries{Manhattan (L1)}} distance.
\item Neighborhood\+: All 3$^\wedge$N -\/ 1 offsets are considered (axis-\/aligned and diagonal moves).
\item Costs\+: Axis-\/aligned steps cost 1.\+0; diagonal/off-\/axis steps use the L2 magnitude of the step vector.
\end{DoxyItemize}

\begin{DoxyAttention}{Attention}
Heuristic admissibility depends on the move set. If diagonal moves are allowed (as in this implementation), Manhattan distance is {\bfseries{not}} strictly admissible unless you adjust step costs accordingly. Prefer {\bfseries{Chebyshev}} (L∞) or Euclidean (L2 with consistent step costs) for diagonal-\/enabled neighborhoods.
\end{DoxyAttention}
