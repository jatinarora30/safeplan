{\footnotesize\ttfamily def sdf\+\_\+astar.\+SDFAStar.\+safety\+Heuristics (\begin{DoxyParamCaption}\item[{}]{self,  }\item[{}]{node,  }\item[{}]{parent }\end{DoxyParamCaption})}



Local SDF-\/based safety measure at a node, relative to the motion direction. 

Uses the precomputed SDF ({\ttfamily self.\+distance\+Transform}) and its gradient ({\ttfamily self.\+grad}). If the SDF at the node is negative, the location is inside an obstacle and the safety measure is set to an infinite penalty.

Otherwise, the method estimates how the SDF changes in the direction of travel from the parent to the node. This directional change is normalized by the largest absolute gradient component at that location to keep the value in a comparable range.
