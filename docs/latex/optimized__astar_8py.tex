\doxysubsection{Detailed Description}
Optimized A$\ast$ planner for N-\/dimensional occupancy grids. 

Implements an A$\ast$ planner with additional path-\/quality and safety refinements on an N-\/D Num\+Py occupancy grid where value 0 denotes free space and value 1 denotes an obstacle. The planner accepts start and goal grid indices and returns a collision-\/free path if one exists.

Key features beyond a standard A$\ast$\+:
\begin{DoxyItemize}
\item Turn-\/aware cost\+: adds a penalty based on local change in heading to discourage sharp turns (see turn\+Heuristics()).
\item Safety inflation\+: uses a distance transform to nudge path points away from nearby obstacles (see safe\+Inflate()).
\item Path shortcutting\+: performs forward and backward line-\/of-\/sight pruning to reduce unnecessary waypoints (see directional\+Optimize() and bidirectional\+Optimize()).
\item Local edge collision checking\+: samples points along a segment and rejects it if any samples fall out of bounds or on obstacles within a safety radius (see is\+Edge\+Free()).
\end{DoxyItemize}

Heuristic and neighborhood generation are provided by heuristics() and adjacent\+Coordinates(). Ensure the heuristic is compatible with the allowed moves and step costs.

\begin{DoxyNote}{Note}

\begin{DoxyItemize}
\item Coordinates are handled in grid index space. Steered or optimized points are rounded to integer indices before validation.
\item The distance transform is computed once per plan call and reused in collision checks and safety inflation.
\item The neighborhood includes all immediate offsets except the zero vector.
\item Only axis-\/aligned moves are expanded in the inner loop of plan() to keep turning penalties predictable; adjust as needed for your benchmark.
\end{DoxyItemize}
