\doxysubsection{Detailed Description}
A$\ast$-\/style planner with a blended optimality + safety heuristic. 

\mbox{\hyperlink{classupp_1_1UPP}{UPP}} biases search toward safer regions by adding a precomputed safety potential to a goal-\/directed distance heuristic. The potential is the FFT convolution of the obstacle mask with a 1/(d\+\_\+inf+ε) kernel over Chebyshev radius R, capturing multi-\/scale obstacle influence.

{\footnotesize\ttfamily def upp.\+UPP.\+plan (\begin{DoxyParamCaption}\item[{}]{self,  }\item[{}]{start,  }\item[{}]{goal,  }\item[{}]{grid }\end{DoxyParamCaption})}



Plan a path from {\ttfamily start} to {\ttfamily goal} over {\ttfamily grid} using \mbox{\hyperlink{classupp_1_1UPP}{UPP}}. 

\begin{DoxyVerb}   @details
   Algorithm flow:
     1. Validate inputs and trivial cases (invalid cells, obstacles, equality).
     2. Precompute the safety potential via @ref precomputeSafety().
     3. Initialize open set with the start node (g=0, f=h_UPP(start,goal)).
     4. Pop the node with smallest f. If it is the goal, stop.
     5. For each valid, free neighbor, compute tentative g using step cost
        (axis=1.0, diagonal=Euclidean), and push with f = g + h_UPP.
     6. Reconstruct path via recorded parents if goal reached.

   @param start tuple[int, ...]
          Start grid cell.
   @param goal tuple[int, ...]
          Goal grid cell.
   @param grid numpy.ndarray
          N-D occupancy grid with {0=free, 1=obstacle}.

   @return success int
           1 if a path was found, else 0.
   @return path list[tuple[int, ...]]
           Path from start to goal (inclusive) if found; else empty.
   @return info list[str]
           Diagnostics about failures or special cases.

   @throws None
           Errors are handled via diagnostics and return values.
\end{DoxyVerb}

{\footnotesize\ttfamily def upp.\+UPP.\+precompute\+Safety (\begin{DoxyParamCaption}\item[{}]{self }\end{DoxyParamCaption})}



Build the safety potential map via FFT convolution. 

\begin{DoxyVerb}   @details
   Let `obs` be the binary obstacle mask (1 where grid==1, else 0).
   Construct a kernel \f$K\f$ on a \f$(2R+1)^N\f$ window with entries
   \f$K(u) = 1/(d_\infty(u)+\epsilon)\f$ for offsets with
   \f$1 \le d_\infty(u) \le R\f$ and \f$K(0)=0\f$.
   The safety map is `S = fftconvolve(obs, K, mode="same")`, clamped to ≥ 0
   and stored as `float32` in `self.preSafety`.

   @post `self.preSafety` is allocated with the same shape as `self.grid`.
\end{DoxyVerb}


\begin{DoxyVerb}Compute safety = sum over obstacles within Chebyshev radius R of 1/(d_inf + epsilon)
using FFT convolution. Stores float32 map in self.preSafety with same shape as self.grid.
\end{DoxyVerb}
