\doxysubsection{Detailed Description}
FS-\/\+Planner\+: Fast and Safe path planning with SDF guidance and Lazy Theta$\ast$. 

Implements FS-\/\+Planner over an N-\/D Num\+Py occupancy grid (0 = free, 1 = obstacle). FS-\/\+Planner augments A$\ast$ with\+:
\begin{DoxyItemize}
\item Signed Distance Field (SDF) based safety-\/aware edge costs,
\item SDF-\/derivative guidance to prefer safer corridors,
\item Lazy Theta$\ast$ line-\/of-\/sight (LOS) parent connections to reduce expansions,
\item Adaptive neighbor pruning toward a blended safety/goal direction.
\end{DoxyItemize}

Grid cells are addressed in index space. Start and goal are N-\/D integer tuples.

Key behaviors\+:
\begin{DoxyItemize}
\item Neighborhood\+: full 3$^\wedge$N -\/ 1 offsets (axis-\/aligned and diagonals).
\item Heuristic\+: Euclidean distance to the goal.
\item Step cost\+: geometric distance, plus an SDF-\/based clearance penalty.
\item Collision checking along edges via uniform samples.
\end{DoxyItemize}

FS-\/\+Planner trades strict shortest-\/path optimality for higher clearance and smoother, lower-\/risk routes. Tuning {\ttfamily cw} adjusts this trade-\/off.

\begin{DoxyItemize}
