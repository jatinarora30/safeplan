\doxysubsection{Detailed Description}
Universal Path Planner (UPP) for N-\/dimensional occupancy grids. 

Implements a heuristic-\/guided grid planner that blends two distance measures and a precomputed safety potential. The environment is an N-\/D Num\+Py occupancy grid where value 0 denotes free space and value 1 denotes an obstacle.

Key ideas\+:
\begin{DoxyItemize}
\item Combined distance heuristic\+: mix of Manhattan and Chebyshev distances.
\item Safety potential\+: fast convolution-\/based map that increases cost near obstacles.
\item Standard open-\/set search using a priority queue with cumulative path cost.
\end{DoxyItemize}
