\doxysubsection{Detailed Description}
Universal Path Planner (UPP) with direction-\/based adaptive safety. 

This planner blends geometric distance and safety potential for N-\/D occupancy grids. It adaptively adjusts the safety weighting β based on {\itshape signed progress} toward the goal.

Key ideas\+:
\begin{DoxyItemize}
\item Combined distance heuristic\+: mix of Manhattan and Chebyshev distances.
\item Safety potential\+: convolution-\/based obstacle cost field.
\item Adaptive β\+: increases when the planner moves closer to the goal, decreases when moving away or stalling.
\item Priority queue A$\ast$ search accumulating total path cost.
\end{DoxyItemize}
