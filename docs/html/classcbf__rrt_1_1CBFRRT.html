Detailed Description

RRT planner with Control Barrier Function (CBF) safety checks.

The planner maintains a tree of collision-free grid indices. Each iteration: 1) Sample a random node with goal bias (see getRandomNode()). 2) Find the nearest existing node in the tree (see getNearestNode()). 3) Steer from the nearest node toward the random sample by at most the step size. 4) Accept the new node only if both the edge is collision-free (see isEdgeFree()) and the CBF safety condition passes (see isSafe()). 5) If the new node is close enough to the goal and the final edge is free, declare success and reconstruct the path.

Members
  • maxIter : int — maximum iterations
  • goalSampleRate : float — probability to sample the exact goal
  • stepSize : float — maximum extension length per expansion
  • pointSamples : int — samples per edge for collision tests
  • gamma1, gamma2 : float — CBF coefficients
  • success : int — 1 on success, else 0
  • info : list[str] — diagnostic messages
  • path : list[tuple[int, ...]] — planned path when found
  • dimension : int — grid dimensionality (set in plan())
  • sizes : list[int] — grid extents per dimension (set in plan())
  • start, goal, grid — problem definition set in plan()

def cbf_rrt.CBFRRT.getObstacleCentersRadii (   self)

Find centers and radii of connected obstacle components.

Labels connected components of obstacle cells. For each component: the center is the mean of its cell indices (centroid), and the radius is the largest Euclidean distance from this centroid to any cell in the component. These values provide a coarse approximation of obstacle geometry for use in the CBF check.

(numpy.ndarray, numpy.ndarray) An array of centers with shape (num_components, dimension) and an array of radii with shape (num_components,).

def cbf_rrt.CBFRRT.isSafe (   self,
  cell,
  vel 
)

Control Barrier Function (CBF) safety check for a candidate cell and velocity.

Uses the obstacle component centers and radii computed by getObstacleCentersRadii(). For each obstacle, a barrier value is formed from the squared distance to the obstacle center minus the squared radius. A change rate is computed using the steer vector. Two user-supplied parameters (gamma1 and gamma2) combine these quantities into a safety measure. If the measure indicates increasing safety or adequate safety margin for any obstacle component, the candidate state is considered safe for insertion into the tree.

Parameters
celltuple[int, ...] Candidate grid cell.
velsequence[float] Steer vector used to reach the candidate cell.
Returns
bool True if the CBF condition passes; otherwise False.
Note
This is an approximate, grid-based CBF that uses component centroids and radii.
