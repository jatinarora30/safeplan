Detailed Description

FS-Planner: Fast and Safe path planning with SDF guidance and Lazy Theta*.

Implements FS-Planner over an N-D NumPy occupancy grid (0 = free, 1 = obstacle). FS-Planner augments A* with:

  • Signed Distance Field (SDF) based safety-aware edge costs,
  • SDF-derivative guidance to prefer safer corridors,
  • Lazy Theta* line-of-sight (LOS) parent connections to reduce expansions,
  • Adaptive neighbor pruning toward a blended safety/goal direction.

Grid cells are addressed in index space. Start and goal are N-D integer tuples.

Key behaviors:

  • Neighborhood: full 3^N - 1 offsets (axis-aligned and diagonals).
  • Heuristic: Euclidean distance to the goal.
  • Step cost: geometric distance, plus an SDF-based clearance penalty.
  • Collision checking along edges via uniform samples.

FS-Planner trades strict shortest-path optimality for higher clearance and smoother, lower-risk routes. Tuning cw adjusts this trade-off.

Inputs (to plan())
  • start : tuple[int, ...] — start grid cell
  • goal : tuple[int, ...] — goal grid cell
  • grid : numpy.ndarray (N-D), occupancy {0=free, 1=obstacle}
Constructor Args
  • pointSamples : int — samples per edge for collision checks
  • cw : float — clearance weight in edge cost
  • epsilon : float — small constant for numerical stability
  • maxNeigh : int — cap on neighbors kept after adaptive pruning
Outputs (from plan())
  • success : int — 1 if a path is found, else 0
  • path : list[tuple[int, ...]] — start→goal (inclusive) or empty
  • info : list[str] — diagnostics (e.g., "Invalid goal")
Note
  • Heuristic optimality guarantees do not strictly hold with clearance penalties.
  • Set cw=0 to approximate baseline A* (still with LOS smoothing if unmodified).
  • Distance transform is computed on (1 - grid) so free cells have positive distance.
