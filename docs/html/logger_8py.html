Detailed Description

Log path-planning experiments (environment, evaluation, and path data) to JSON files.

Creates a per-run directory structure and writes one JSON file per iteration and algorithm. Each file captures:

  • Algorithm name
  • Evaluation metrics (as provided by the caller)
  • Start/goal pair
  • Path info (sequence, diagnostics)
  • Environment info (grid, cell size, environment name and description)

Directory layout:

  • <outputDir>/<runDetails>/<algo>/iter_<N>.json
Note
  • Directories are created on demand with basic error handling.
  • The grid is serialized via tolist() for JSON compatibility.
  • The success flag (if present in pathData[0]) is not currently written (line kept commented).
