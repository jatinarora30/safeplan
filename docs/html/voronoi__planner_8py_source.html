1 """
2 @file voronoi_planner.py
3 @brief Voronoi-roadmap planner for N-dimensional occupancy grids.
4 
5 @details
6 Builds a roadmap from the Voronoi diagram of obstacle clusters and searches it
7 with A* to obtain a collision-free path between a start and a goal cell on an
8 N-D NumPy occupancy grid (0 = free, 1 = obstacle). The planner first checks
9 whether the straight line from start to goal is collision-free; if so, it returns
10 that two-point path. Otherwise, it:
11  1) segments obstacles via connected-component labeling and computes each
12  component's centroid as a Voronoi site,
13  2) constructs the Voronoi diagram (finite edges only),
14  3) builds a graph from collision-free Voronoi edges,
15  4) connects start and goal to their K nearest Voronoi vertices if collision-free,
16  5) runs A* on this graph to produce a piecewise-linear path.
17 
18 @par Inputs
19 - @p start : tuple[int, ...] — start grid cell (for example, (row, col))
20 - @p goal : tuple[int, ...] — goal grid cell
21 - @p grid : numpy.ndarray (N-D), values {0=free, 1=obstacle}
22 
23 @par Outputs
24 - @p success : int — 1 if a path is found, else 0
25 - @p path : list[tuple[int, ...]] — sequence of cells from start to goal (inclusive)
26 - @p info : list[str] or str — diagnostics (for example, "Invalid goal") or a success note
27 
28 @note
29 - Requires a sufficient number of obstacle components to form a meaningful
30  Voronoi diagram (this implementation expects more than four).
31 - Collision checks on edges sample @ref pointSamples evenly spaced points and
32  round to the nearest grid indices for occupancy testing.
33 - Grid convention: 1 = obstacle, 0 = free.
34 
35 @see BasePlanner
36 
37 @references
38 - SciPy Voronoi: https://docs.scipy.org/doc/scipy/reference/generated/scipy.spatial.Voronoi.html
