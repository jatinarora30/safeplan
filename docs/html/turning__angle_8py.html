Detailed Description

Turning Angle evaluator for planning algorithms.

Implements Turning Angle evaluation based upon the path provided on the grid. It uses the third derivate of position with step size of 1

Inputs
  • start : tuple[int, ...] — start grid cell (e.g., (row, col))
  • goal : tuple[int, ...] — goal grid cell
  • grid : numpy.ndarray (N-D), values {0=free, 1=obstacle}
  • cellSize : Size of cell(in m) for real world computation
  • Path : Path given by planner to evaluate metrices
Outputs
  • val: Turning Angle evaluated
