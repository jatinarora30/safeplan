Detailed Description

RRT planner operating on N-D occupancy grids.

The planner maintains a tree of explored, collision-free grid indices. Each iteration: 1) Sample a random node with goal bias (see getRandomNode()). 2) Find the nearest existing node in the tree (see getNearestNode()). 3) Steer from the nearest node toward the random sample by at most the step size. 4) If the edge is collision-free (see isEdgeFree()), add the new node. 5) If the new node is close enough to the goal and the final edge is free, declare success and reconstruct the path.

Members
  • maxIter : int — maximum iterations
  • goalSampleRate : float — probability to sample the exact goal
  • stepSize : float — maximum extension length per expansion
  • pointSamples : int — samples per edge for collision tests
  • success : int — 1 on success, else 0
  • info : list[str] — diagnostic messages
  • path : list[tuple[int, ...]] — planned path when found
  • dimension : int — grid dimensionality (set in plan())
  • sizes : list[int] — grid extents per dimension (set in plan())
  • start, goal, grid — problem definition set in plan()

Sample a random node in the grid with goal bias.

With probability equal to the goal sampling rate, returns the goal directly. Otherwise, uniformly samples an integer index within the grid bounds along each dimension.

Check if the segment between two points is collision-free.

Samples a fixed number of points linearly between the endpoints. Each sample is rounded to the nearest grid index. The edge is rejected if any sampled index is out of bounds or lies on an obstacle cell.

Parameters

pt2sequence[float] Segment end in continuous coordinates.
