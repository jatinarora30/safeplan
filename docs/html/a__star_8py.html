Detailed Description

A* planner for N-dimensional occupancy grids.

Implements A* (A-star) search over an N-D NumPy occupancy grid where 0 denotes free space and 1 denotes an obstacle. The planner takes a start and goal index (tuples of ints) and returns a collision-free path if one exists. The heuristic and neighborhood definition are provided by heuristics() and adjacentCoordinates() and must be chosen consistently.

Inputs
  • start : tuple[int, ...] — start grid cell (e.g., (row, col))
  • goal : tuple[int, ...] — goal grid cell
  • grid : numpy.ndarray (N-D), values {0=free, 1=obstacle}
Outputs
  • success : int — 1 if a path is found, else 0
  • path : list[tuple[int, ...]] — sequence of cells from start→goal (inclusive)
  • info : list[str] — diagnostics (e.g., "Invalid goal")
Note
Ensure the heuristic matches the move set:
  • 4-neighbors → Manhattan distance
  • 8-neighbors / diagonal moves → Chebyshev (or Euclidean with proper step costs)
