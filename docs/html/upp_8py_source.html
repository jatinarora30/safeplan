3 @brief Universal Path Planner (UPP) for N-dimensional occupancy grids.

6 Implements a heuristic-guided grid planner that blends two distance measures

8 grid where value 0 denotes free space and value 1 denotes an obstacle.
9 
10 Key ideas:
11 - Combined distance heuristic: mix of Manhattan and Chebyshev distances.
12 - Safety potential: fast convolution-based map that increases cost near obstacles.
13 - Standard open-set search using a priority queue with cumulative path cost.
14

132  @brief Build the safety potential map via FFT convolution.
133  @details
134  Creates a kernel whose weight at each offset is the inverse of the
135  Chebyshev distance plus a small constant, within a given radius. The
136  obstacle mask is convolved with this kernel to produce a safety map
