def optimized_astar.OptimizedAStar.calculateRasterCoeffiecient (   self)

Compute a raster coefficient over the axis-aligned box between start and goal.

Extracts the hyper-rectangle spanning start and goal, counts obstacle cells, and returns the natural logarithm of the obstacle ratio. The value is clamped away from zero and one for numerical stability.

Returns
float Natural log of the clamped obstacle ratio in the rectangle.

  pt2 
)

Check if the segment between two points is collision-free with clearance.

Samples a fixed number of points linearly between the endpoints. Each sample is rounded to the nearest grid index. The edge is rejected if any sampled index is out of bounds, lies on an obstacle, or has distance-to-obstacle below the configured safety radius.

Parameters
pt1sequence[float] Segment start in continuous coordinates.
pt2sequence[float] Segment end in continuous coordinates.
Returns
bool True if all sampled points are valid, free, and safely clear.

  • path (list[tuple[int, ...]]) : sequence from start to goal (inclusive)
  • info (list[str]) : diagnostics or failure reasons
Diagnostics
- "Invalid start", "Invalid goal"
- "Start has obstacle", "Goal has obstacle"
- "Start and goal are same"
- "Failed to reconstruct path."
Note
The method computes a raster coefficient for environmental clutter, uses an A* loop with turn-aware total cost, inflates the path for safety, and finally applies bidirectional shortcutting.

◆ safeInflate()

def optimized_astar.OptimizedAStar.safeInflate (   self)

Push intermediate waypoints away from obstacles using a distance transform.

Computes a distance transform on the free-space mask and attempts to replace intermediate path points with nearby points that have larger clearance, up to a configured target radius. The start and goal are preserved.

Returns
list[tuple[int, ...]] Updated path after safety inflation.
