5 @details
6 Implements Dijkstra’s shortest-path search on an N-D NumPy occupancy grid where
7 value 0 denotes free space and value 1 denotes an obstacle. The planner accepts
8 start and goal grid indices and returns a collision-free path with minimum
9 accumulated cost if one exists.
10 
11 This implementation expands a full 3^N - 1 neighborhood (axis-aligned and diagonal
12 moves). Axis-aligned steps use unit cost; diagonal or off-axis steps use the
13 Euclidean length of the step vector. A priority queue orders expansions by the
14 current best distance from the start. No heuristic is used.
15 
16 Neighborhood generation is provided by @ref adjacentCoordinates().
17 
18 @par Inputs
19 - @p start : tuple[int, ...] — start grid cell (for example, (row, col))

21 - @p grid : numpy.ndarray (N-D), values {0=free, 1=obstacle}
22 
23 @par Outputs
24 - @p success : int — 1 if a path is found, else 0
25 - @p path : list[tuple[int, ...]] — sequence of cells from start to goal (inclusive)
26 - @p info : list[str] — diagnostics (for example, "Invalid goal")
27
