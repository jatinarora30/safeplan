Run Weighted A* from start to goal on the provided grid.

Parameters
starttuple[int, ...] Start index in the N-D grid.
goaltuple[int, ...] Goal index in the N-D grid.
gridnumpy.ndarray Occupancy grid (0=free, 1=obstacle).
Returns
tuple
  • success (int) : 1 if a path is found, else 0
  • path (list[tuple[int, ...]]) : sequence from start to goal (inclusive)
  • info (list[str]) : diagnostics or failure reasons
Diagnostics
- "Invalid start", "Invalid goal"
- "Start has obstacle", "Goal has obstacle"
- "Start and goal are same"
- "Failed to reconstruct path."
Note
Uses priority g + w·h. With weights greater than 1 the algorithm is greedier and may return a suboptimal path more quickly.
