1 """
2 @file sdf_astar.py
3 @brief Signed Distance Field (SDF) guided A* planner for N-dimensional occupancy grids.
4 
5 @details
6 Implements an A* variant that uses a Signed Distance Field (SDF) to bias search
7 toward safer regions. The environment is an N-D NumPy occupancy grid where value
8 0 denotes free space and value 1 denotes an obstacle. The planner accepts start
9 and goal grid indices and returns a collision-free path if one exists.
10 
11 Key ideas:
12 - A standard grid A* framework: priority queue, cumulative path cost, and a
13  neighborhood that includes all immediate offsets.
14 - An SDF-based safety term: a distance transform and its gradient encourage
15  exploration toward areas with better clearance (see @ref safetyHeuristics()).
16 - A combined priority: mixes travel distance, path cost so far, and a local
17  safety measure based on the SDF gradient along the current motion direction
18  (see @ref combinedHeuristics()).
19 
20 Heuristic and neighborhood generation are provided by @ref heuristics() and
21 @ref adjacentCoordinates(). Ensure the heuristic is compatible with the allowed
22 moves and step costs.
23 
24 @par Constructor Arguments
25 - @p k1 : float — weight on path-cost term inside the combined priority
26 - @p k2 : float — weight on heuristic distance term inside the combined priority
27 
28 @par Inputs (to @ref plan())
29 - @p start : tuple[int, ...] — start grid cell (for example, (row, col))
30 - @p goal : tuple[int, ...] — goal grid cell
31 - @p grid : numpy.ndarray (N-D), values {0=free, 1=obstacle}
32 
33 @par Outputs
34 - @p success : int — 1 if a path is found, else 0
35 - @p path : list[tuple[int, ...]] — sequence of cells from start to goal (inclusive)
36 - @p info : list[str] — diagnostics (for example, "Invalid goal")
37 
38 @note
39 - Coordinates are handled in grid index space.
40 - The SDF is computed once per plan call using a distance transform on the free
41  space mask; its gradient is reused in the safety term.
42 - The neighborhood includes all immediate offsets except the zero vector.
43 - The safety term normalizes the SDF gradient change along the current motion
44  direction, with simple clamping to avoid division by zero.
45 
46 @see BasePlanner
47 
48 @references
49 - SDF-guided A*
50  https://ieeexplore.ieee.org/document/10364842/
51 """
