1 """
2 @file inflated_astar.py
3 @brief Inflated A* (clearance-augmented) planner on N-D occupancy grids.
4 
5 @details
6 Implements a grid-based A* that first inflates obstacles by a user-specified
7 clearance radius, then plans on the inflated grid. Inflation is performed via
8 a Euclidean Distance Transform (EDT) computed on free space (1 - grid). Any
9 cell whose EDT value is less than or equal to @p radiusInflate is marked as
10 occupied in the inflated grid. This biases paths to maintain a minimum
11 clearance from obstacles without modifying the A* edge cost itself.
12 
13 Conventions:
14 - Grid values: 0 = free, 1 = obstacle.
15 - Coordinates are integer grid indices in N dimensions.
16 - Neighborhood: all 3^N - 1 offsets (axis-aligned + diagonals).
17 - Step costs: axis-aligned steps cost 1.0; diagonal/off-axis steps use the L2
18  magnitude of the step vector.
19 - Heuristic: Manhattan (L1) by default. For diagonal moves, prefer Chebyshev (L∞)
20  or Euclidean (L2 with consistent edge costs) to preserve admissibility.
21 
22 @par Inputs (to plan())
23 - @p start : tuple[int, ...] — start grid cell (e.g., (row, col[, ...]))
24 - @p goal : tuple[int, ...] — goal grid cell
25 - @p grid : numpy.ndarray (N-D), values {0=free, 1=obstacle}
26 
27 @par Constructor Args
28 - @p radiusInflate : float — minimum clearance radius (in grid cells) enforced by obstacle inflation
29 
30 @par Outputs (from plan())
31 - @p success : int — 1 if a path is found, else 0
32 - @p path : list[tuple[int, ...]] — sequence of cells from start→goal (inclusive) or empty
33 - @p info : list[str] — diagnostics (e.g., "Invalid goal", "Failed to reconstruct path")
34 
35 @note
36 - This method enforces safety by *removing* near-obstacle cells from the search
37  space rather than by penalizing costs. It may render some problems infeasible
38  if @p radiusInflate is too large for narrow passages.
39 - If you keep diagonal moves with Manhattan heuristic, A* is no longer strictly
40  admissible. Use Chebyshev or Euclidean if optimality guarantees matter.
41 
42 @see BasePlanner, distance_transform_edt
43 
44 @references
45 - Inflated A*: https://arxiv.org/pdf/2003.00368
46 """
