Detailed Description

Universal Path Planner blending distance and safety costs.

The planner builds a safety potential via convolution, then runs a grid search that prefers shorter and safer routes. The total priority combines:

  • a weighted sum of Manhattan and Chebyshev distances, and
  • a safety cost that rises near obstacles.

◆ precomputeSafety()

def upp.UPP.precomputeSafety (   self)

Build the safety potential map via FFT convolution.

Creates a kernel whose weight at each offset is the inverse of the Chebyshev distance plus a small constant, within a given radius. The obstacle mask is convolved with this kernel to produce a safety map that increases near obstacles. The result is stored in self.preSafety with the same shape as the grid.

Postcondition
self.preSafety is allocated as float32.
Compute safety = sum over obstacles within Chebyshev radius R of 1/(d_inf + epsilon)
using FFT convolution. Stores float32 map in self.preSafety with same shape as self.grid.
