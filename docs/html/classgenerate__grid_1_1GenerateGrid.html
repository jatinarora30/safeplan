Detailed Description

Generates and caches a discrete occupancy grid with polygonal obstacles.

The class loads a scene configuration (grid size, polygons, etc.), computes half-space representations for polygons, rasterizes them into a grid (1 = occupied, 0 = free), and optionally samples random start/goal pairs lying in free cells. The generated grid is cached to <configName>_grid.json for repeatability and faster subsequent runs.

Note
  • Grid indexing is 0-based across all dimensions.
  • Obstacles are considered "inside" if a cell index (interpreted as a point) satisfies all half-space inequalities for at least one polygon.
  • Vertices are explicitly marked as occupied.

None generate_grid.GenerateGrid.computeHalfspace (   self)

Compute half-space representations (A, b) for all polygons.

Uses scipy.spatial.ConvexHull to obtain the H-representation of each polygon from its vertices. Stores the results into self.A and self.b as parallel lists.

Precondition
self.polygons_data must be loaded via loadJson().
Postcondition
self.A[k], self.b[k] hold the half-space for polygon k.

◆ getmap()

def generate_grid.GenerateGrid.getmap (   self)

Build or load the occupancy grid and associated metadata.

  • If a cached grid file (<configName>_grid.json) exists, load it.
  • Else, load base config, build an N-D zero grid, mark obstacle cells by polygon half-space tests, and (optionally) generate random start/goal pairs in free cells. Then cache to JSON.
Returns
Tuple (grid, cellSize, envName, startGoalPairs):
  • grid (np.ndarray): N-D array of 0 (free) / 1 (occupied).
  • cellSize (float/int): Size of a cell in world units.
  • envName (str): Environment name.
  • startGoalPairs (list): List of dicts with "start" and "goal" tuples.
Exceptions
FileNotFoundErrorIf neither the cache nor the base config exists.
Postcondition
  • self.grid is a numpy array with obstacles marked as 1.
  • self.startGoalPairs is populated (from config or randomly).
  • A JSON cache is written when generating from base config.

Reimplemented from baseenv.BaseEnv.

tuple generate_grid.GenerateGrid.makeConvexHull (   self,
  vertices 
)

Convert a set of polygon vertices to half-space form.

Parameters
verticesIterable of vertex coordinates (shape: N x D).
Returns
(A, b) where A @ x <= b describes the convex hull.

Uses scipy.spatial.ConvexHull to compute plane equations in the form [A | c] so that A x + c = 0 for faces. We rearrange to A x <= b with b = -c.

Warning
Vertices must define a valid (non-degenerate) convex polygon/polytope.
