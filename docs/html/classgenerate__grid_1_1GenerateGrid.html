def generate_grid.GenerateGrid.computeHalfspace (   self)

\brief Build half-space representations (A, b) for all polygons.

\details Uses \c scipy.spatial.ConvexHull to compute \c equations for each polygon, then stores A (all but the last column) and b (negated last column) for use in point-in- polygon tests.

\post After this call, \c self.A and \c self.b each contain one entry per polygon.

◆ getmap()

def generate_grid.GenerateGrid.getmap (   self)

\brief Load or generate the occupancy grid and start/goal pairs.

\details

  • If \c <configFolder>_grid.json exists and is readable, load it and populate members.
  • Otherwise, read the input JSON, build the grid by rasterizing convex polygons, and (optionally) generate random start/goal pairs on free cells, then write the output file.

\returns A tuple (grid, cellSize, envName, startGoalPairs).

  • \c grid : numpy.ndarray of 0/1 occupancy (shape: (gridSize,)*dimension)
  • \c cellSize : float
  • \c envName : str
  • \c startGoalPairs : list of dicts with keys \"start\\" and \"goal\\"

def generate_grid.GenerateGrid.makeConvexHull (   self,
  vertices 
)

\brief Convert polygon vertices to half-space form using a convex hull.

\param vertices List of vertex coordinates (each a list/tuple of length \c self.dimension). \returns A pair (A, b) where A is an array of shape (m, dim) and b has shape (m,). The polygon\'s convex region is represented as A x \le b. \note This treats the input polygon as \b convex; concave inputs are convexified.
