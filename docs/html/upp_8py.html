Detailed Description

Universal Path Planner (UPP) for N-dimensional occupancy grids.

Implements a heuristic-guided grid planner that blends two distance measures and a precomputed safety potential. The environment is an N-D NumPy occupancy grid where value 0 denotes free space and value 1 denotes an obstacle.

Key ideas:

  • Combined distance heuristic: mix of Manhattan and Chebyshev distances.
  • Safety potential: fast convolution-based map that increases cost near obstacles.
  • Standard open-set search using a priority queue with cumulative path cost.
Constructor Arguments
  • alpha : float — blend coefficient between Manhattan and Chebyshev distances
  • beta : float — weight on the safety potential
  • radius : int — Chebyshev radius for safety kernel support
  • epsilon : float — small constant to avoid division by zero in the kernel
Inputs (to @ref plan())
  • start : tuple[int, ...] — start grid cell (for example, (row, col))
  • goal : tuple[int, ...] — goal grid cell
  • grid : numpy.ndarray (N-D), values {0=free, 1=obstacle}
Outputs
  • success : int — 1 if a path is found, else 0
  • path : list[tuple[int, ...]] — sequence of cells from start to goal (inclusive)
  • info : list[str] — diagnostics (for example, "Invalid goal")
Note
  • Coordinates are handled in grid index space.
  • The safety potential is precomputed once per planning call.
  • Neighborhood generation includes all immediate offsets except the zero vector.
